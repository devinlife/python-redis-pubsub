# python-redis-pubsub

## 1. clone git repo

```
$ git clone https://github.com/devinlife/python-redis-pubsub.git
$ cd python-redis-pubsub
```

## 2. synchronous

```
$ cd synchronous
$ docker compose up --build
```

building images and send messages

## 3. asynchronous

```
$ cd asynchronous
$ docker compose up --build
```
